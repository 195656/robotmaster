问题说明：其中当物体快速移动时，会发生识别断帧的问题，观察二值化后的img可发现，是移动过快导致与移动方向垂直的轮廓区域“看不见”（比如拿纸画矩形，矩形的长平行于地面，宽垂直于地面，当纸张面对摄像头做左右移动时候，会发现二值化画面里面的矩形的宽若隐若现导致不能判断为一个封闭轮廓）后续查证是运动模糊导致的轮廓断裂，改进可通过harris测定目标的特征来锁定物体。或结合yolo和opencv来做识别。
