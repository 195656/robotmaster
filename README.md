## 01. 颜色分割 (01_fenge.py)

`01_fenge.py` 文件实现了对目标图片的特征区域分割，通过多次按键可分别查看每次图像处理后的逐步执行结果。

### 核心思路
1. **色彩空间分析**：通过查看图片各通道值，确认红色和粉红色的 H（色调）值通常在 `140-180` 之间，以此作为颜色分割的核心条件。
2. **预处理与优化**：提取 HSV 掩膜后，考虑到粉红色的外部噪声点以及床面内部的空白反光点，采用**腐蚀与膨胀（形态学开闭运算）**进行去噪和孔洞填补。
3. **AI 使用情况**：将图片上传给 AI 分析其 HSV 特征值，并由 AI 协助完成了绝大多数核心分割代码的编写。

## 02. 多边形轮廓检测 (02_duobianxing.py)

### 核心思路
1. **标准流水线**：遵循经典的图像预处理步骤：**去噪声 $\rightarrow$ 二值化 $\rightarrow$ 查找轮廓 $\rightarrow$ 几何筛选**。
2. **开发方式**：纯手搓核心算法逻辑。

## 03. 四角星动态识别 (02_wujiaoxing.py)

### 核心思路
1. **多帧关联**：整体预处理与多边形检测任务一致，但为了解决动态画面中的**移动模糊与闪烁问题**，引入了时间序列缓存。采用 `detection_history.append(len(filtered_contours) > 0)` 记录多帧识别状态，进行平滑判定。
2. **开发方式**：几乎完全复用了上一个任务的基础架构代码，其中针对四角星独特形状（如凹多边形、Hu矩）的筛选逻辑部分由 AI 生成。

## 04. 机器人位姿估计解算 (03_robotmaster)

本模块采用分步递进的策略，从单项识别、内参标定到静态验证，最终实现动态 PnP 位姿解算。

### 核心思路
1. **精确目标识别 (`detect_sq.py`)**：首先独立实现对待识别物体的精准几何轮廓识别，提前排除因目标跟丢而导致的 PnP 解算失败问题。
2. **相机标定与验证 (`03_biaoding.py` / `03_biaodingyanzheng.py`)**：
   * 采用标准的张正友棋盘格标定法获取相机内参矩阵与畸变系数。
   * 标定完成后，先利用棋盘格进行重投影精度测试，彻底排除因相机内参不准带来的三维解算误差。
3. **静态姿态测试**：先在静态照片/单帧画面上运行 PnP 算法，成功测出目标在不同空间角度下的三维坐标（平移与旋转）。
4. **动态实时解算 (`03_robotmaster.py`)**：在上述基础完备后，引入动态视频流与防抖滤波算法，完成最终的实时三维位姿估计任务。
